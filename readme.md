
# India State District 2024

This project contains the list of Indian states and their respective District.


## JSON Format

The json data is available in format:

```json
{
    "updated-by": 2024,
    "states": [
        {
            "stateName": "Andhra Pradesh",
            "abbv": "AP",
            "capital": "Amaravati",
            "distLength": 26,
            "districts": ["Alluri Sitharama Raju", ...]
        }
        ...
    ]
}
```
